引用类型

  • Object类
    方括号的使用，优点是可以通过变量来访问属性。

  • Array类
    Array是javascript用的非常广泛的一个类。JS中数组的每一项，可以保存任何数据类型。而且可以各不相同。

  • 数组的length不是只读的，它可以被设置改变

  • Array各种可调用的函数：
    • 由于多框架可能会干扰到instanceof,所以改用.isArray(value)
    • 栈方法： pop,push
    • 队列方法：push,shift
    • 排序：sort,reverse
    • concat：基于当前数组新建数组；slice，创建一个基于当前数组去掉某项；splice：可以有多应用。splice(第一项，项数，替换值)因此可实现删除，插入，替换的功能
    • 位置方法：indexOf(),lastIndexOf()

但是值得注意的事sort()方法调用了toString()，所以在比较的时候，会出现10在5前边，所以解决方法是自己编写compare

  • 五个迭代方法:

    • every:对每一项迭代，全部为true,才返回true
    • filter：过滤，返回那些true的
    • forEach:对每一项给定函数，没有返回值
    • map：对每一项给定函数，全部返回值
    • some:每一项给定参数，只要有一个true，则返回true

  • 归并方法reduce，四个参数，前一个值，当前值，项的索引，数组对象

  • Date类型

  • Date.now()取得当前时间，用于简化代码工作，也可以使用一元加操作，+new Date()

  • RegExp类型

  • 结合正则表达式学习理解这部分
    pattern:g,全局；i：不区分大小写；m：多行模式；

  • Function类型

  • 没有重载，后边的函数覆盖前边的函数

JavaScript 复习笔记

今天又从头开始看JavaScript红宝书，这一次本着更加细致的原则，从头去再刷一遍，另外犀牛书也可以开始看了。

HTML中的JS元素

  • 注意两个属性：

    • defer 立即下载，延迟执行。使用方式defer = ‘defer’
    • async 下载后，顺序可能改变。使用方式 ‘async’

  • HTML中尽量不要使用

不符合规范，所以在平时写作时候，尽量保持良好的书写规范。

  • 平稳退化，当不支持脚本或禁用脚本时候调用。

  • Javascript中区分大小写，标识符书写规范采用驼峰型，第一个字母用小写后边的每个单词首字母大写。例：firstChild myBlog

  • ; 不是必须的，但是鉴于书写规范，推荐保持使用的习惯。

  • 关键字，保留字，不可以作为标识符，具体谷歌。

  • var操作符定义的变量为局部变量，函数退出后即销毁，直接创建，则为全局变量，但可能导致不必要的麻烦。

  • null == undefined返回的是true。

  • 数据类型转换成Boolean时，TRUE or False

    • string 非空 “”
    • Number 非零 0或者NaN
    • object 任何对象 null
    • undefined N/A undefined

  • NaN表示非数值，与任何的值不同，NaN与自己也不相同，0除以0时候是NaN

  • 位移操作 >> 和 >>>
    前者是有符号的移动，后者是无符号的移动

  • 全等和不全等

  • ‘label’标签功能
    ‘with’语句

  • 没有重载，不存在签名特性
